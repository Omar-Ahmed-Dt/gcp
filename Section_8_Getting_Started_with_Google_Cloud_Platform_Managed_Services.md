| Service Model | Full Name                   | You Manage                            | Provider Manages                     | Best For                          | GCP Examples                   |
| ------------- | --------------------------- | ------------------------------------- | ------------------------------------ | --------------------------------- | ------------------------------ |
| IaaS          | Infrastructure as a Service | OS, runtime, middleware, app, scaling | VM, networking, storage              | Full control, legacy apps         | Compute Engine                 |
| PaaS          | Platform as a Service       | Application code                      | OS, runtime, infrastructure, scaling | Web apps, APIs, rapid development | App Engine                     |
| CaaS          | Container as a Service      | Containers, deployment configs        | Cluster infrastructure               | Microservices, DevOps workflows   | GKE (Google Kubernetes Engine) |
| FaaS          | Function as a Service       | Function code only                    | Everything else (infra + scaling)    | Event-driven workloads            | Cloud Functions                |
| DBaaS         | Database as a Service       | Data, schema, queries                 | Database engine, backups, scaling    | Managed databases                 | Cloud SQL, Firestore, BigQuery |
